## Обзор

Этот проект генерирует высококачественные синтетические табличные данные, сохраняя статистические и структурные свойства исходного набора данных. Он позволяет надежно расширять данные для приложений машинного обучения, предоставляя ценный инструмент для обмена данными с соблюдением конфиденциальности и улучшения обучения моделей в условиях ограниченности реальных данных.

## Основные функции

1. **Реализация WGAN-GP**: Реализует вагнеровскую GAN с градиентным штрафом (WGAN-GP) для стабильного обучения за счёт использования вагнеровского расстояния и обеспечения липшицевой непрерывности через градиентный штраф, что приводит к более надёжной и согласованной генерации синтетических данных.
2. **Сохранение статистики мета-признаков (MFS)**: Улучшает GAN за счёт сохранения статистических свойств исходных данных через статистику мета-признаков (MFS), включая среднее, дисперсию, корреляцию, ковариацию, собственные значения и моменты более высокого порядка, обеспечивая высокую точность синтетических табличных данных.
3. **Вычисление MFS на основе PyTorch**: Предоставляет полностью дифференцируемую реализацию извлечения мета-признаков, нативную для PyTorch (через MFEToTorch), что позволяет выполнять оптимизацию на основе градиентов «от начала до конца» и интеграцию с циклом обучения GAN для выравнивания MFS в реальном времени.
4. **Вагнеровское расстояние для выравнивания MFS**: Использует оптимальную транспортировку (вагнеровское расстояние) для измерения и минимизации расхождения между распределениями мета-признаков реальных и синтетических данных, улучшая статистическую схожесть и полезность сгенерированных выборок.
5. **Гибкие архитектуры генератора и дискриминатора**: Поддерживает настраиваемые архитектуры нейронных сетей для генератора и дискриминатора с использованием остаточных блоков и настраиваемых размеров слоёв, что позволяет адаптироваться к различной сложности данных и потребностям моделирования.
6. **Комплексные метрики оценки**: Включает набор метрик полезности и точности, таких как R², RMSE, MAPE, расхождение Дженсена — Шеннона, расстояние между корреляционными матрицами и топологический анализ данных, для строгой оценки качества синтетических данных.
7. **Отслеживание экспериментов с помощью Aim**: Интегрируется с Aim для отслеживания экспериментов в реальном времени, регистрируя потери при обучении, градиенты, сгенерированные образцы и метрики оценки, обеспечивая воспроизводимость и анализ производительности между запусками обучения.
8. **Генератор на основе остаточных сетей**: Использует архитектуру генератора на основе остаточных соединений, которая объединяет входы по слоям, что способствует созданию более глубоких сетей и улучшает поток градиентов во время обучения для более эффективной генерации синтетических данных.
9. **Настраиваемый конвейер обучения**: Предоставляет модульную и настраиваемую конфигурацию обучения через гиперпараметры, такие как скорость обучения, размер пакета, эпохи, вес градиентного штрафа и вес потерь MFS, обеспечивая точный контроль над процессом обучения.

## Установка

Установите GAN_MFS2 одним из следующих способов:

**Сборка из исходников:**

1. Склонируйте репозиторий GAN_MFS2:
```sh
git clone https://github.com/fl1pcoin/GAN_MFS2
```

2. Перейдите в каталог проекта:
```sh
cd GAN_MFS2
```

### 1. Установка зависимостей

```bash
pip install -r req.txt
```
## Начало работы

Чтобы начать работу с GAN_MFS2, выполните следующие шаги для обучения модели WGAN-GP, улучшенной с помощью статистики мета-признаков (MFS), на вашем табличном наборе данных.

### 2. Подготовка данных

Убедитесь, что ваши данные представлены в табличном формате (например, CSV) с целевым столбцом. Пример:

```python
import pandas as pd
import numpy as np
from sklearn.model_selection import train_test_split

# Загрузка данных
data = pd.read_csv('your_data.csv')
y = data.pop('target').values
X = data.values

# Разделение и объединение для обучения
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)
training_data = np.hstack([X_train, y_train.reshape(-1, 1)])
```

### 3. Настройка и обучение модели

Используйте `TrainerModified` для обучения с улучшенной MFS:

```python
from wgan_gp.training import TrainerModified
from wgan_gp.models import Generator, Discriminator

# Определение гиперпараметров
learning_params = {
    'epochs': 1000,
    'learning_rate_D': 0.0001,
    'learning_rate_G': 0.0001,
    'batch_size': 64,
    'gp_weight': 10,
    'generator_dim': (64, 128, 64),
    'discriminator_dim': (64, 128, 64),
    'mfs_lambda': [0.1, 2.0],
    'subset_mfs': ['mean', 'var', 'eigenvalues'],
    'sample_number': 10,
    'sample_frac': 0.5,
}

# Инициализация тренера и запуск обучения
trainer = TrainerModified(
    generator=Generator(**learning_params),
    discriminator=Discriminator(**learning_params),
    **learning_params
)
trainer.train(data_loader, epochs=1000, plot_freq=100)
```

### 4. Мониторинг обучения с помощью Aim

Проект поддерживает отслеживание экспериментов через Aim. Чтобы включить:

```python
import aim
tracker = aim.Run(experiment="WGAN-GP")
tracker["hparams"] = learning_params
```

### 5. Оценка синтетических данных

После обучения оцените синтетические данные с помощью встроенных метрик:

- **Полезность**: R², RMSE, MAPE
- **Точность**: Расстояние между корреляционными матрицами, расхождение Дженсена — Шеннона

См. `utils.py` для функций оценки.

Для получения дополнительной информации соберите документацию локально:

```bash
mkdocs serve --config-file osa_mkdocs.yml
```

Затем перейдите по адресу `http://localhost:8000`.
